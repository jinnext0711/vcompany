# vCompany デザインルール

## デザイン方針

- **スタイリッシュですっきりとしたデザイン**
- 図表を適度に盛り込み、視覚的に情報を伝える
- 文字を詰め込まず、余白を活かす
- 1スライドに伝えるメッセージは1つ

## 禁止事項

1. **デフォルトフォントの使用禁止**
   - Inter, Arial, Calibri, MS Gothic 等のデフォルトフォントは使用しない
   - 指定フォントのみを使用すること
2. **無駄なアニメーションの使用禁止**
   - 装飾目的のアニメーションは一切使用しない
3. **グラデーションの使用禁止**
   - 装飾目的でのグラデーションは使用しない
   - 背景・図形・テキストすべてにおいてフラットカラーを使用する
4. **アイコンの過剰使用禁止**
   - 地図マーク、設定アイコン、歯車マーク等の汎用アイコンを装飾的に使わない
   - アイコンは情報伝達に必要な場合のみ、最小限に使用する

## レイアウトルール

### スライド共通
- スライドサイズ: 16:9（ワイドスクリーン）
- 上下余白: 0.8インチ以上
- 左右余白: 0.8インチ以上
- コンテンツ間の余白: 0.3インチ以上

### テキスト
- 1スライドの箇条書きは最大5項目
- 1項目は1〜2行以内
- 本文テキストは1スライドあたり最大60文字 x 5行
- 箇条書きの行頭記号は「●」または「—」（シンプルなもの）

### 図表
- 図表は情報を補完するために積極的に使用する
- フラットカラーで統一（グラデーション禁止）
- グラフの配色はカラーパレットに準拠する
- 図表にはキャプションを付ける

### スライド構成パターン
1. **タイトルスライド** — タイトル + サブタイトル、中央配置
2. **見出し + 箇条書き** — 最も基本的なレイアウト
3. **見出し + 図表** — 図表中心のスライド
4. **2カラム** — 左右に情報を並列配置
5. **数値ハイライト** — KPIや重要数値を大きく表示
6. **まとめスライド** — キーメッセージを3つ以内で提示
