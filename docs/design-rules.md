# vCompany デザインルール

## デザイン方針

- **スタイリッシュですっきりとしたデザイン**
- 図表を適度に盛り込み、視覚的に情報を伝える
- 文字を詰め込まず、余白を活かす
- **1スライドに伝えるメッセージは1つ**（One Message Per Slide）
- 装飾ではなく情報伝達を目的としたデザインにする
- データの視覚表現は正確に、歪みなく行う

## 禁止事項

1. **デフォルトフォントの使用禁止**
   - Inter, Arial, Calibri, MS Gothic, MS PGothic, Times New Roman 等は使用しない
   - 指定フォント（Noto Sans JP）のみを使用すること
2. **無駄なアニメーションの使用禁止**
   - 装飾目的のアニメーションは一切使用しない
3. **グラデーションの使用禁止**
   - 装飾目的でのグラデーションは使用しない
   - 背景・図形・テキストすべてにおいてフラットカラーを使用する
4. **アイコンの過剰使用禁止**
   - 地図マーク、設定アイコン、歯車マーク等の汎用アイコンを装飾的に使わない
   - アイコンは情報伝達に必要な場合のみ、最小限に使用する
5. **チャートジャンクの禁止**（Tufte原則）
   - 3D効果、装飾的グリッド線、不要なラベル、背景画像をグラフに使用しない
   - データの理解に貢献しない視覚要素は全て排除する
6. **色のみによる情報伝達の禁止**（WCAG準拠）
   - 色だけに依存した情報の区別をしない
   - 形状、テキスト、パターン等を併用して情報を伝える

## アクセシビリティ基準（WCAG AA準拠）

### コントラスト比

| レベル | 通常テキスト | 大きいテキスト（18pt以上 / 太字14pt以上） |
|--------|------------|--------------------------------------|
| **AA（必須）** | **4.5:1** 以上 | **3:1** 以上 |
| **AAA（推奨）** | **7:1** 以上 | **4.5:1** 以上 |

- UIコンポーネント（枠線、区切り線等）: **3:1** 以上
- vCompany の配色コントラスト比:
  - `#1A1A2E`（Primary）on `#FFFFFF`（白背景）→ **15.4:1** ✓ AAA
  - `#2D2D2D`（Text）on `#FFFFFF`（白背景）→ **12.6:1** ✓ AAA
  - `#53868B`（Highlight）on `#FFFFFF`（白背景）→ **3.9:1** ✓ AA（大テキスト）
  - `#6B6B6B`（Light Text）on `#FFFFFF`（白背景）→ **5.0:1** ✓ AA

### アクセシビリティチェックリスト
- [ ] 全テキストがコントラスト比 AA 基準を満たしている
- [ ] 色だけに依存した情報伝達をしていない
- [ ] テキストのリサイズに対応している
- [ ] 図表にキャプション・代替テキストがある

## カラーパレット

運用カラーは **4色**（Primary / Highlight / Text / Background）を基本とし、
補助カラーは必要に応じて使用する。

### 基本カラー（常用4色）

| 用途 | カラー | コード | 使用場面 |
|------|--------|--------|---------|
| Primary | 濃紺 | `#1A1A2E` | ヘッダー、見出し、強調、背景（反転時） |
| Highlight | シアン | `#53868B` | アクセント、リンク、数値強調、区切り線 |
| Text | ダークグレー | `#2D2D2D` | 本文テキスト |
| Background | ホワイト | `#FFFFFF` | スライド・ページ背景 |

### 補助カラー（必要時のみ使用）

| 用途 | カラー | コード | 使用場面 |
|------|--------|--------|---------|
| Secondary | スレートブルー | `#16213E` | サブヘッダー、補助要素 |
| Light Text | ミディアムグレー | `#6B6B6B` | キャプション、補足テキスト |
| Light BG | ライトグレー | `#F5F5F5` | セクション背景、カード |
| Border | ボーダーグレー | `#E0E0E0` | 区切り線、枠線 |

### グラフ・図表用カラー（最大4色）

| 順序 | コード | 用途 |
|------|--------|------|
| 1 | `#1A1A2E` | 主要データ |
| 2 | `#53868B` | 比較データ |
| 3 | `#6B6B6B` | 補助データ |
| 4 | `#E0E0E0` | 背景・参考データ |

## タイポグラフィ

### フォント

フォントファミリーは **Noto Sans JP の1種類のみ**に統一する。
ウェイト（太さ）で階層を表現する。

### プレゼンスライド用サイズ

| 用途 | サイズ | ウェイト | 行間 | 備考 |
|------|-------|---------|------|------|
| スライドタイトル | 36pt | Bold (700) | 1.2倍 | ライブ発表を考慮し拡大 |
| セクション見出し | 28pt | Bold (700) | 1.2倍 | |
| サブ見出し | 20pt | Medium (500) | 1.3倍 | |
| 本文 | 18pt | Regular (400) | 1.5倍 | 24pt推奨だが配布資料寄りで18pt |
| キャプション | 12pt | Regular (400) | 1.5倍 | |
| 数値ハイライト | 40pt | Bold (700) | 1.0倍 | KPI等の強調表示 |

### ドキュメント（Markdown / HTML / PDF）用サイズ

| 用途 | サイズ | ウェイト | 行間 |
|------|-------|---------|------|
| タイトル (h1) | 32px | Bold (700) | 1.2倍 |
| 見出し (h2) | 24px | Bold (700) | 1.3倍 |
| サブ見出し (h3) | 18px | Medium (500) | 1.3倍 |
| 本文 (p) | 16px | Regular (400) | 1.8倍 |
| キャプション | 12px | Regular (400) | 1.5倍 |

### テキストルール
- 1行あたりの文字数: 最大 **40文字**（日本語）/ **75文字**（英語）
- 段落間スペース: フォントサイズの **2倍** 以上
- テキスト配置: **左揃え**（均等揃え禁止）

## 情報設計の原則

### 1スライド1メッセージ
- 各スライドで伝えるメッセージは **1つだけ** に絞る
- スライドタイトルがそのメッセージを表す
- 箇条書き・図表はメッセージの根拠として配置する

### テキスト量の制限
- 1スライドの箇条書きは **最大5項目**
- 1項目は **1〜2行以内**
- 本文テキストは1スライドあたり **最大3行**
- 箇条書きの行頭記号は **「—」**（ダッシュ）で統一

### 視覚的階層（Visual Hierarchy）
- **サイズ**: 重要な情報ほど大きく
- **太さ**: 見出しはBold、本文はRegular
- **色**: Primaryは重要要素、Light Textは補足
- **配置**: 重要な情報は左上に（F字パターン）

## データビジュアライゼーション原則（Tufte準拠）

### データインク比（Data-Ink Ratio）
- **目標: 1.0に近づける**
- グラフ上の全ての視覚要素がデータを表現しているか確認する
- 排除すべき: 背景画像、3D効果、装飾的グリッド線、不要なラベル

### ライファクター（Lie Factor）
- **許容範囲: 0.95〜1.05**
- グラフの視覚表現がデータの実際の比率を正確に反映していること
- 面積・体積の比較で誤解を招く表現をしない

### グラフ作成ルール
- グラフの種類はデータに適したものを選択する
  - 比較 → 棒グラフ
  - 推移 → 折れ線グラフ
  - 構成比 → 円グラフ（項目数5以下の場合のみ）
  - 相関 → 散布図
- Y軸は原則 **0から開始** する（切り取りによる誤解を防ぐ）
- ラベル・凡例はグラフに直接配置する（離れた位置に置かない）
- グリッド線は **薄いグレー（#E0E0E0）** で最小限に

### スモールマルチプル
- 複数のデータセットを比較する場合、同一形式の小さなグラフを並列配置する
- 軸のスケールは全グラフで統一する

## スライド構成パターン

| パターン | 用途 | レイアウト |
|---------|------|-----------|
| **1. タイトル** | 表紙 | タイトル + サブタイトル、左寄せ |
| **2. セクション区切り** | 章の切り替え | セクション名を大きく中央配置 |
| **3. 見出し + 箇条書き** | 標準コンテンツ | 上部に見出し、下部に箇条書き（最大5項目） |
| **4. 見出し + 図表** | データ提示 | 上部に見出し、中央に図表 |
| **5. 2カラム** | 比較・対比 | 左右に情報を並列配置 |
| **6. 数値ハイライト** | KPI・重要指標 | 大きな数値（40pt）+ 補足テキスト |
| **7. まとめ** | 結論 | キーメッセージを3つ以内で提示 |
